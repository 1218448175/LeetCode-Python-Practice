## [1. 两数之和](https://leetcode.cn/problems/two-sum/)

### 中等

给定一个整数数组 `nums` 和一个整数目标值 `target`，请你在该数组中找出 **和为目标值** *`target`*  的那 **两个** 整数，并返回它们的数组下标。

你可以假设每种输入只会对应一个答案，并且你不能使用两次相同的元素。

你可以按任意顺序返回答案。

**示例 1：**

<pre><strong>输入：</strong>nums = [2,7,11,15], target = 9
<strong>输出：</strong>[0,1]
<strong>解释：</strong>因为 nums[0] + nums[1] == 9 ，返回 [0, 1] 。
</pre>

---

## 题解：哈希表单次遍历与“寻找另一半”策略

### 1. 核心思路：从“寻找”变为“记忆”

两数之和的最优解法是将问题的复杂度从 $O(N^2)$（双重循环枚举）降低到 $O(N)$。其核心思想是利用哈希表（Dictionary）记录已经遍历过的数字及其对应的索引。

- **匹配逻辑**：当我们遍历到数字 `num` 时，我们并不关心它本身，而是关心它的“另一半”——即 `target - num` 是否在之前出现过。

- **即时检查**：如果“另一半”已经在哈希表中，说明我们找到了一对和为 `target` 的组合，直接返回结果。

- **动态更新**：如果没找到，就将当前的数字和索引存入哈希表，作为后续数字的“潜在匹配对象”。

---

### 2. 执行逻辑的详细拆解

#### A. 初始化哈希表

创建一个空的 `hashtable = dict()`。在这个映射关系中：

- **Key（键）**：存储数组中的数值。

- **Value（值）**：存储该数值对应的索引 `i`。

#### B. 线性扫描与匹配

使用 `enumerate` 同时获取数组的索引 `i` 和数值 `num`：

1. **计算差值**：计算 `complement = target - num`。

2. **查表**：在哈希表中检索 `complement`。
   
   - **命中**：说明之前有一个数等于 `complement`。根据题目要求返回 `[hashtable[complement], i]`。由于我们是按顺序遍历的，`hashtable[complement]` 一定小于 `i`。
   
   - **未命中**：将当前数字存入表中：`hashtable[num] = i`。

#### C. 边界处理

由于题目保证“每种输入只会对应一个答案”，因此在循环内部一定会触发 `return`。最后的 `return []` 仅作为代码完备性的兜底。

---

### 3. 逻辑亮点：边走边记

这种方法的精妙之处在于**不需要预处理整个哈希表**：

- **规避重复使用**：通过“先检查再存入”的顺序，完美避免了“同一个元素使用两次”的情况（例如 `target` 为 6，数组第一个数是 3，如果先存再找，3 会和自己匹配）。

- **一次遍历**：每个元素只被访问一次，空间和时间达到了完美的平衡。

---

### 4. 复杂度分析

- **时间复杂度**：$O(N)$。其中 $N$ 是数组的长度。我们只需遍历一遍数组，且哈希表的查询和插入操作平均耗时均为 $O(1)$。

- **空间复杂度**：$O(N)$。在最坏情况下（匹配对象在数组末尾），我们需要在哈希表中存储 $N-1$ 个元素。

---

### 代码回顾

Python

```
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
        # 1. 构造哈希表用于存放 {数值: 索引}
        hashtable = dict()

        for i, num in enumerate(nums):
            # 2. 寻找当前数字的“匹配对象”
            complement = target - num

            # 3. 如果匹配对象已在表中，说明找到答案
            if complement in hashtable:
                return [hashtable[complement], i]

            # 4. 否则，将当前数字及其索引存入表，供后续数字匹配
            hashtable[num] = i

        return []
```
