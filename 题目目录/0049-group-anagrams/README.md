## [49. 字母异位词分组](https://leetcode.cn/problems/group-anagrams/)

### 中等

给你一个字符串数组，请你将 **字母异位词** 组合在一起。可以按任意顺序返回结果列表。

**示例 1:**

<div class="example-block">
<p><strong>输入:</strong> strs = ["eat", "tea", "tan", "ate", "nat", "bat"]</p>

<p><strong>输出: </strong>[["bat"],["nat","tan"],["ate","eat","tea"]]</p>

<p><strong>解释：</strong></p>

<ul>
    <li>在 strs 中没有字符串可以通过重新排列来形成 <code>"bat"</code>。</li>
    <li>字符串 <code>"nat"</code> 和 <code>"tan"</code> 是字母异位词，因为它们可以重新排列以形成彼此。</li>
    <li>字符串 <code>"ate"</code> ，<code>"eat"</code> 和 <code>"tea"</code> 是字母异位词，因为它们可以重新排列以形成彼此。</li>
</ul>
</div>

<p><strong>提示：</strong></p>

<ul>
    <li><code>1 <= strs.length <= 10<sup>4</sup></code></li>
    <li><code>0 <= strs[i].length <= 100</code></li>
    <li><code>strs[i]</code> 仅包含小写字母</li>
</ul>

---

## 题解：字符串标准化排序与哈希分类策略

### 1. 核心思路：寻找“异位词”的共同签名

字母异位词的本质是字母相同但顺序不同。要将它们分到同一组，关键在于找到一个**共同的键（Key）**。

你的代码采用了一种非常直观且高效的**标准化策略**：

- **排序作为指纹**：对于任何一组字母异位词（如 `"eat"`, `"tea"`, `"ate"`），如果我们将它们内部的字符按字典序重新排序，它们都会变成同一个字符串（`"aet"`）。

- **分类聚合**：利用哈希表（`defaultdict`）将这个排序后的字符串作为“组标签”，将原始字符串存入对应的列表中。

---

### 2. 执行逻辑的详细拆解

#### A. 初始化哈希桶

使用 `collections.defaultdict(list)`。这种容器的优点是：当访问一个不存在的键时，它会自动创建一个空列表，省去了手动判断键是否存在的繁琐逻辑。

#### B. 标准化处理（排序）

遍历字符串数组中的每个字符串 `st`：

1. **字符重组**：执行 `sorted(st)`。这会将字符串拆解为字符列表并排序。

2. **生成键值**：执行 `"".join(...)`。将排序后的列表重新拼接成字符串。这个字符串就是该词所属组的“唯一身份证”。

#### C. 归类与导出

1. **按键入库**：将原始字符串 `st` 添加进 `mp[key]` 对应的列表中。

2. **结果转换**：遍历结束后，`mp` 里的每个 `value` 都是一组异位词集合。最后通过 `list(mp.values())` 将其转化为题目要求的列表嵌套列表格式。

---

### 3. 逻辑亮点：标准化思想

这种解法的精妙之处在于将“判断两个词是否为异位词”的 $O(N)$ 逻辑，转化为了“将单词归入其唯一标识符”的映射逻辑。

- **简洁性**：代码极其精简，逻辑链条非常短。

- **健壮性**：无论异位词如何排列，排序后的结果是唯一的，这从数学上保证了分类的准确性。

---

### 4. 复杂度分析

- **时间复杂度**：$O(N \cdot K \log K)$。其中 $N$ 是字符串数组的大小，$K$ 是字符串的最大长度。我们需要遍历 $N$ 个字符串，而每个字符串的排序耗时为 $O(K \log K)$。

- **空间复杂度**：$O(N \cdot K)$。主要开销是存储所有字符串的哈希表。

---

### 代码回顾

```python
class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        # 1. 创建默认值为列表的字典，用于存放分组结果
        mp = collections.defaultdict(list)

        for st in strs:
            # 2. 标准化：排序后的字符串是异位词的“唯一签名”
            key = "".join(sorted(st))

            # 3. 将原始单词归类到对应签名的“桶”中
            mp[key].append(st)

        # 4. 返回所有桶的内容
        return list(mp.values())
```
