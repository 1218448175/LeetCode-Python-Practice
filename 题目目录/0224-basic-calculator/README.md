## [224. 基本计算器](https://leetcode.cn/problems/basic-calculator/)

### 中等

给你一个字符串表达式 `s` ，请你实现一个基本计算器来计算并返回它的值。

注意:不允许使用任何将字符串作为数学表达式计算的内置函数，比如 `eval()` 。

**示例 1：**

**输入**：s = "1 + 1"
**输出**：2

---

## 题解：符号展开与栈式符号管理策略

### 1. 核心思路：将括号展开为全局符号

这道“困难”级别的题目难点在于括号的嵌套以及负号对括号内所有项的影响。例如：`1 - (2 + 3)` 实际上等于 `1 - 2 - 3`。

你的代码采用了一种非常精妙的**符号展开法**：

- **不处理优先级，只处理正负号**：由于题目只有加减法，我们可以通过维护一个“当前环境下的实际正负号”来消除括号。

- **符号栈 (`ops`)**：栈顶元素表示当前括号层级外部的总符号。

- **实时转换**：将每一个数字根据其前面的符号以及它所在括号层级的总符号，直接转化为加法或减法累加到结果中。

---

### 2. 执行逻辑的详细拆解

#### A. 符号栈的含义

`ops = [1]` 初始状态。栈里存的是**作用于当前层级的全局符号**。

- 当遇到 `-(...)` 时，括号内的所有正号变负号，负号变正号。

- 栈顶元素 `ops[-1]` 始终代表当前位置对应的“实际极性”。

#### B. 符号更新规则

1. **遇到 `+`**：当前数字的符号取决于括号外的整体符号。`sign = ops[-1]`。

2. **遇到 `-`**：当前数字的符号与括号外的符号相反。`sign = -ops[-1]`。

3. **遇到 `(`**：进入新层级。将当前的 `sign` 压入栈中。这代表接下来括号内的内容都要受这个 `sign` 的控制。

4. **遇到 `)`**：退出层级。`ops.pop()`，恢复到上一层的符号环境。

#### C. 数字处理

使用 `while` 循环解析多位数字，计算出 `num` 后，直接执行 `ret += num * sign`。此时的 `sign` 已经包含了所有嵌套括号累积的正负变换。

---

### 3. 示例模拟分析

对于表达式：`1 - (3 - 5)`

1. 读取 `1`：`sign=1`, `ret=1`。

2. 读取 `-`：`sign = -ops[-1] = -1`。

3. 读取 `(`：`ops.append(-1)` -> `ops = [1, -1]`。

4. 读取 `3`：当前内层 `sign` 为父级的 `-`。`ret = 1 + (3 * -1) = -2`。

5. 读取下一个 `-`：`sign = -ops[-1]` (此时栈顶是 -1) -> `sign = 1`。

6. 读取 `5`：`ret = -2 + (5 * 1) = 3`。

7. 读取 `)`：`ops.pop()`。

最终结果 `3`，等价于 `1 - 3 + 5`。

---

### 4. 复杂度分析

- **时间复杂度**：$O(N)$。字符串只被扫描一次，栈操作均为 $O(1)$。

- **空间复杂度**：$O(N)$。在最坏情况下（全是括号嵌套），栈的深度可达 $N$。

---

### 5. 代码回顾

```python
class Solution:
    def calculate(self, s: str) -> int:
        ops = [1]
        sign = 1

        ret = 0
        n = len(s)
        i = 0
        while i < n:
            if s[i] == ' ':
                i += 1
            elif s[i] == '+':
                sign = ops[-1]
                i += 1
            elif s[i] == '-':
                sign = -ops[-1]
                i += 1
            elif s[i] == '(':
                ops.append(sign)
                i += 1
            elif s[i] == ')':
                ops.pop()
                i += 1
            else:
                num = 0
                while i < n and s[i].isdigit():
                    num = num * 10 + ord(s[i]) - ord('0')
                    i += 1
                ret += num * sign
        return ret

```
