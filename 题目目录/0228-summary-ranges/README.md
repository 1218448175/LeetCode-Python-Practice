## [228. 汇总区间](https://leetcode.cn/problems/summary-ranges/)

### 简单

给定一个  **无重复元素** 的 **有序** 整数数组 `nums` 。

区间 `[a,b]` 是从 `a` 到 `b`（包含）的所有整数的集合。

返回 ***恰好覆盖数组中所有数字** 的 **最小有序** 区间范围列表* 。也就是说，`nums` 的每个元素都恰好被某个区间范围所覆盖，并且不存在属于某个区间但不属于 `nums` 的数字 `x` 。

列表中的每个区间范围 `[a,b]` 应该按如下格式输出：

- `"a->b"` ，如果 `a != b`
- `"a"` ，如果 `a == b`

**示例 1：**

<pre><strong>输入：</strong>nums = [0,1,2,4,5,7]
<strong>输出：</strong>["0->2","4->5","7"]
<strong>解释：</strong>区间范围是：
[0,2] --> "0->2"
[4,5] --> "4->5"
[7,7] --> "7"
</pre>

---

## 题解：双指针线性扫描与连续区间分组策略

### 1. 核心思路：利用“有序”特征进行断点探测

本题的关键在于输入是一个**无重复且有序**的数组。这意味着在一个连续区间 $[a, b]$ 内，任何相邻两个元素的差值必须恰好为 $1$。

你的代码采用了经典的**双指针（或快慢指针）分块**思想：

- **锚定起点**：用 `low` 记录当前连续区间的起始位置。

- **探测终点**：用 `high` 向后探索，只要满足 `nums[high + 1] == nums[high] + 1`，就说明区间还在延续。

- **格式化输出**：一旦发现断点（差值不为 1 或到达数组末尾），就根据 `low` 和 `high` 是否重合来决定输出格式。

---

### 2. 执行逻辑的详细拆解

#### A. 外层循环：跳跃式遍历

`while i < n` 并不是逐个步进的，而是直接“跳跃”到下一个区间的起点。这种设计能有效减少不必要的逻辑判断。

#### B. 内层循环：边界扩张

`while high < n - 1 and nums[high + 1] == nums[high] + 1`：

- 这个循环负责找出当前能连成的最长序列。

- 由于数组是有序的，我们只需要比较相邻元素，不需要回溯或建立复杂的映射。

#### C. 条件分支与格式化

- **单点区间**：如果 `high == low`，说明这个数字是孤立的，直接转为字符串 `f"{nums[low]}"`。

- **多点区间**：如果 `high > low`，使用 `->` 连接首尾值。

#### D. 索引更新

`i = high + 1`：直接将指针移至下一个未处理元素的起始位置，保证了每个元素只被访问一次。

---

### 3. 逻辑亮点：简洁的边界控制

代码中 `high < n - 1` 的判断非常关键，它优雅地处理了数组越界问题，确保了在访问 `nums[high + 1]` 时始终是安全的。同时，利用 Python 的 f-string 进行字符串拼接，使代码既整洁又高效。

---

### 4. 复杂度分析

- **时间复杂度**：$O(n)$。虽然存在嵌套循环，但指针 `i` 和 `high` 始终是向右移动的，每个元素仅被扫描一次。

- **空间复杂度**：$O(1)$。除了存储结果的列表外，只使用了常数级别的额外空间（不计入结果列表的情况下）。

---

### 代码回顾

Python

```
class Solution:
    def summaryRanges(self, nums: List[int]) -> List[str]:
        n = len(nums)
        i = 0
        ans = []

        while i < n:
            low = i
            high = i
            # 探测连续的“梯子”
            while high < n - 1 and nums[high + 1] == nums[high] + 1:
                high += 1

            # 根据区间长度决定显示格式
            if high == low:
                ans.append(f"{nums[low]}")
            else:
                ans.append(f"{nums[low]}->{nums[high]}")

            # 跨越已处理的区间
            i = high + 1

        return ans
```
