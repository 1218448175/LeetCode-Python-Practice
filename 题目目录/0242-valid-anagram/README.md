## [242. 有效的字母异位词](https://leetcode.cn/problems/valid-anagram/)

### 简单

给定两个字符串 `s` 和 `t` ，编写一个函数来判断 `t` 是否是 `s` 的 字母异位词。

**示例 1:**

<pre><strong>输入:</strong> s = "anagram", t = "nagaram"
<strong>输出:</strong> true
</pre>

---

## 题解：哈希频次对比与集合差集校验策略

### 1. 核心思路：字母构成的“指纹”匹配

字母异位词的本质是：**两个字符串包含的字母种类完全相同，且每种字母出现的频率也完全一致**。就像是同一组零件拼成了不同的形状。

你的代码利用了 Python 中 `collections.Counter` 的强大特性：

- **词频指纹**：通过 `Counter` 将字符串转化为“字母:频率”的映射。

- **对称差集校验**：通过执行两次减法并进行逻辑判断，来确保两个“指纹”是完全重合的。只有当 `s` 的资源能覆盖 `t`，且 `t` 的资源也能完全覆盖 `s` 时，两者才互为异位词。

---

### 2. 执行逻辑的详细拆解

#### A. 频次统计

代码首先为两个字符串分别创建了计数器 `s_cnt` 和 `t_cnt`。这一步会将字符串转化为类似于 `{'a': 3, 'n': 1, ...}` 的结构。

#### B. 双向覆盖判定

在 Python 中，`Counter` 的减法运算（`s_cnt - t_cnt`）会保留那些“在被减数中多出来的”部分：

1. **`s_cnt - t_cnt`**：如果结果不为空，说明 `s` 中某些字母的频率高于 `t`，或者含有 `t` 没有的字母。

2. **`t_cnt - s_cnt`**：如果结果不为空，说明 `t` 中某些字母的频率高于 `s`。

3. **逻辑或运算 (`or`)**：只要其中任何一个差集不为空，`or` 的结果就会是一个非空的 `Counter` 对象（在 Python 中布尔值为 `True`）。

#### C. 最终取反

最后通过 `not` 对运算结果取反：

- 如果两个差集都为空（即完全相等），`not False` 返回 `True`。

- 只要有任何不匹配，`not True` 返回 `False`。

---

### 3. 逻辑亮点：利用 Python 特性的极简设计

这种写法极具 Pythonic 风格：

- **语义清晰**：代码非常直观地表达了“既要 `s` 包含 `t`，也要 `t` 包含 `s`”的逻辑。

- **规避长度检查**：虽然通常我们会先判断 `len(s) != len(t)` 来快速剪枝，但你的双向减法逻辑在数学上已经内含了长度和内容的一致性校验。

---

### 4. 复杂度分析

- **时间复杂度**：$O(N)$。其中 $N$ 是字符串的长度。我们需要各遍历一遍 `s` 和 `t` 来建立计数器，哈希表的减法操作耗时与字符集大小有关（常数级）。

- **空间复杂度**：$O(K)$。$K$ 为字符集的大小。由于题目规定仅包含小写字母，哈希表的大小上限为 26，空间开销是稳定的常数级别。

---

### 代码回顾

```python
class Solution:
    def isAnagram(self, s: str, t: str) -> bool:
        # 1. 建立频率映射
        s_cnt = Counter(s)
        t_cnt = Counter(t)

        # 2. 只有两个方向的差集都为空，才说明两者完全相同
        # 这种写法巧妙利用了 Counter 相减只保留正频次的特性
        return not ((s_cnt - t_cnt) or (t_cnt - s_cnt))
```
